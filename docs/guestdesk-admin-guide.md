# GuestDesk: guia rápido do admin

## Objetivo
Este guia é para quem opera o painel no dia a dia.

## Onde começa
1. Entrar em `/login`
2. Fazer login
3. Abrir `/admin`

## O que cada módulo faz
### Dashboard
Serve para:

1. ter visão geral
2. acessar módulos rápido
3. revisar próximos passos

### Informações do hotel
Aqui você ajusta:

1. nome do hotel
2. cidade
3. horários
4. Wi-Fi
5. links oficiais
6. logo

### Serviços
Aqui você publica:

1. cards informativos
2. serviços do hotel
3. links de apoio
4. botões de acesso

### Departamentos
Aqui você publica:

1. canais de contato
2. setores do hotel
3. links de atendimento
4. horários de atendimento

### Políticas
Aqui você publica:

1. regras do hotel
2. orientações para hóspedes
3. políticas operacionais

## Regras práticas de operação
1. Escreva sempre em português como idioma principal
2. Salve conteúdos curtos, claros e objetivos
3. Revise links antes de publicar
4. Teste o resultado na página pública depois de mudanças grandes

## Tradução no admin
### Como funciona
1. O conteúdo fonte é salvo em PT
2. O sistema tenta gerar EN e ES automaticamente
3. O admin mostra status por item

### O que significa cada status
1. `Tradução completa`: EN e ES disponíveis
2. `Tradução parcial`: um idioma disponível e outro faltando
3. `Apenas português`: nenhuma tradução salva ainda

### Quando usar “Retraduzir”
Use quando:

1. o item ficou com status parcial
2. a tradução falhou
3. você atualizou o conteúdo e quer forçar nova tentativa

## Cuidados operacionais
1. Não apagar itens sem confirmar impacto na página pública
2. Não publicar links quebrados
3. Não deixar textos muito longos sem testar mobile
4. Validar tradução de conteúdos importantes

## Rotina recomendada
1. revisar dashboard
2. revisar itens com tradução parcial
3. validar links
4. testar página pública
