# LibGuest: descrição oficial do produto

LibGuest é um diretório digital para hotéis com painel administrativo, experiência pública otimizada para celular e publicação multilíngue com português como idioma fonte.

LibGuest é uma solução digital criada para organizar a comunicação do hotel com mais clareza, sofisticação e praticidade. A plataforma combina um painel administrativo simples de operar com uma página pública pensada para o hóspede, reunindo serviços, departamentos, políticas e informações essenciais em uma experiência visual premium.

LibGuest ajuda hotéis a substituir materiais dispersos e fluxos manuais por uma apresentação digital mais organizada, elegante e fácil de atualizar.
