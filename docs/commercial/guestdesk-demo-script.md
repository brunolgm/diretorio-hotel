# LibGuest: roteiro de demo

## Objetivo
Mostrar o LibGuest como um produto pronto para uso comercial, destacando clareza operacional, experiência do hóspede e facilidade de atualização.

## Abertura sugerida
`O LibGuest é um diretório digital para hotelaria que organiza a experiência de informação do hóspede e facilita a atualização do conteúdo pelo time do hotel.`

## Sequência recomendada
1. mostrar landing e contexto do produto
2. mostrar login
3. mostrar dashboard
4. mostrar gestão de hotel
5. mostrar serviços, departamentos e políticas
6. mostrar experiência pública
7. mostrar idiomas e analytics

## Fechamento sugerido
`O LibGuest combina apresentação premium com operação simples. Ele funciona bem como produto de demonstração, entrega comercial e base real para uso em hotelaria.`
