# LibGuest: playbook de novo hotel

## Objetivo
Subir e validar um novo hotel no LibGuest com o menor risco possível.

## Passo a passo
1. criar hotel no banco
2. definir slug
3. definir subdomínio quando aplicável
4. vincular perfil administrativo ao hotel
5. preencher informações principais
6. cadastrar serviços, departamentos e políticas
7. validar experiência pública

## Validação mínima
1. rota por slug funcionando
2. subdomínio funcionando quando configurado
3. botões principais funcionando
4. idioma padrão em português
5. traduções EN/ES salvas quando houver conteúdo traduzido

## Boas práticas
- começar com conteúdo simples e consistente
- revisar contatos e links finais
- validar experiência no celular
- confirmar identidade visual e logo

## Naming
- produto atual: `LibGuest`
- domínio operacional atual: `guestdesk.digital`
